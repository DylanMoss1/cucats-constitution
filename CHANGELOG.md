# Changelog

## 2025-10-18

### Changed

-   Job Titles: Refactored for clarity and consistency
    -   Renamed "Webmaster / System Administrator" to "System Administrator"
    -   Removed "(≥1)" notation from Frontend Developer, Platform Engineer, and Operations Engineer titles

## 2025-10-16

### Added

-   Executive Committee: Added detailed role descriptions for all positions
    -   President: Oversees general management, planning, and sponsor relations
    -   Vice President: Assists President, manages day-to-day Committee operations, organizes socials
    -   Junior Treasurer: Manages invoices, grants, bank account, and financial transactions
    -   Secretary: Records minutes, manages venue bookings, and handles communications
-   Socials Subcommittee: New section defining social event management
    -   Socials Director: Manages and runs Society socials
    -   Socials Officer: Per year group with £1,000 annual budget
-   Events Subcommittee: New section for non-social events
    -   Events Director: Manages and runs Society events
    -   Events Officer: Organizes computer science focused events
    -   Publicity Officer: Promotes events through social media and creates promotional materials
-   Technology Subcommittee: Formalized technology roles structure
    -   Technology Director: Oversees technical direction
    -   System Administrator: Maintains internal systems and websites
    -   Frontend Developer: Ensures user-friendly and capable websites
    -   Platform Engineer: Builds and maintains core event systems
    -   Operations Engineer: Develops tools to improve efficiency

### Changed

-   Candidate Eligibility: Extended full-time undergraduate/master's requirement to all roles except Senior Treasurer (previously only Executive Committee)
